### Sol
a를 `int`형으로 입력받고, b는 변환해주지 않은 상태인 문자열 그대로 둔다.  
이것은 문자열의 인덱스를 이용해 출력하기 위함이다.
```python
a = int(input()) # int형으로 변환
b = input() # 문자열 그대로 둠

# 문자열의 인덱스를 이용해 숫자 맨 뒤부터 곱하면서 출력한다.
print(a * int(b[2]))
print(a * int(b[1]))
print(a * int(b[0]))
print(a * int(b))
```
